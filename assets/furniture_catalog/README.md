# Local furniture catalog

The preferred professional catalog is stored in [`pro/`](pro/). It contains a
curated 1K glTF subset from [Poly Haven](https://polyhaven.com/) with real
textured geometry for furniture, lighting, plants, mirrors, framed art, clocks,
and styling objects. Poly Haven publishes these assets under CC0.

The GLB files in this folder are the compact
[Kenney Furniture Kit](https://kenney.nl/assets/furniture-kit) fallback used
where the professional open catalog does not have a matching kitchen or
bathroom fixture. They are also CC0.

The walkthrough normalizes every object to real room dimensions, keeps a
stable floor pivot, preserves authored material variation, coordinates it with
the user's palette, automatically stages the apartment, and supports rotation.
No catalog item is generated from or projected from a furniture screenshot.

See `LICENSE.txt` and `pro/LICENSE.txt` for source and license links.
