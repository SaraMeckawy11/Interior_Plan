# Local furniture catalog

These native GLB models are selected from [Kenney's Furniture Kit](https://kenney.nl/assets/furniture-kit) and distributed under CC0.

They are used as clean, transformable furniture components in the 3D walkthrough. The application applies measured dimensions, a stable floor pivot, room-palette materials, automatic designer placement, and user-controlled rotation. They are not generated from or projected from furniture screenshots.
