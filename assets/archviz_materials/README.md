# Local architectural material library

The walkthrough ships with local 1K PBR material packages from
[ambientCG](https://ambientcg.com/). Each asset includes its color, normal,
roughness and displacement maps; packages that provide ambient occlusion keep
that map as well.

Installed materials:

- WoodFloor007 — warm oak flooring
- WoodFloor041 — dark/rustic wood flooring
- Plaster001 — clean interior plaster
- Wallpaper002C — softly patterned wall finish
- Tiles133A — clean modern bathroom/kitchen tile
- Marble019 — light natural marble
- Carpet001 — woven rug/carpet
- Fabric062 — plain rough woven upholstery and curtain fabric
- Fabric083 — optional check-pattern woven fabric
- Concrete034 — smooth architectural concrete
- Terrazzo019M — beige terrazzo flooring

All ambientCG assets are released under the
[Creative Commons CC0 license](https://ambientcg.com/index.php?menu=license).
They are bundled locally so opening or regenerating a walkthrough never
depends on a paid service or an internet connection.
